# Assignment 2 ER Diagrams

This folder contains the regenerated ER diagram assets for the updated Ghost Drop database schema.

## Files

- `generate_er_diagrams.py` - source used to build the diagrams
- `ghostdrop_er_basic.gv` - Graphviz source for the basic ER diagram
- `ghostdrop_er_basic.png` - rendered basic ER diagram
- `ghostdrop_er_basic.pdf` - rendered basic ER diagram
- `ghostdrop_er_formal.gv` - Graphviz source for the formal ER diagram
- `ghostdrop_er_formal.png` - rendered formal ER diagram
- `ghostdrop_er_formal.pdf` - rendered formal ER diagram

## Schema covered

The diagrams reflect the current database design in `Ghost_Drop/backend/sql/init_schema.sql` and include the updated Module B `portfolio_entries` table along with the vault, token, file, session, audit, CAPTCHA, and expiry relationships.

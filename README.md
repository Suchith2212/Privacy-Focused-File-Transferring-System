# CS432 Assignment Repository

This repository is organized for GitHub upload and submission review.

The repository contains both `Ass1/` and the consolidated `Ass2/` submission package.

## Repository Link

GitHub Repository: https://github.com/Suchith2212/Privacy-Focused-File-Transferring-System

## Primary Entry Points

- `Ass1/`
- `Ass2/`

## Inside `Ass2`

- `CS432_Track1_Submission/` final packaged submission for Track 1
- `Project_432/` BlindDrop project source and project-level documentation
- `deliverables/` assignment documents, reports, and submission bundle
- `Ass1/` Assignment 1 materials
- `Privacy-Focused-File-Transferring-System/` related prototype/reference project

## Recommended Reading Order

1. `Ass1/README.md`
2. `Ass2/CS432_Track1_Submission/README.md`
3. `Ass2/Project_432/README.md`
4. `Ass2/deliverables/README.md`

## GitHub Upload

Use the following commands from the repository root:

```powershell
git add .
git commit -m "Add Ass1 and Ass2 submission files"
git push origin main
```

If your branch is not `main`, replace it with your branch name.

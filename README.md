# CS432 Assignment Repository

This repository is organized for GitHub upload, submission review, and local development.

## Repository Link

GitHub Repository: https://github.com/Suchith2212/Privacy-Focused-File-Transferring-System

## Primary Entry Points

- `Project_432/` active BlindDrop project source at the repository root
- `Project_Assignments/Assignment1/` Assignment 1 materials and documentation
- `Project_Assignments/Assignment2/` Assignment 2 package, submission bundle, and project copy
- `Project_Assignments/Assignment3/` reserved Assignment 3 folder

## Inside `Project_Assignments/Assignment2`

- `CS432_Track1_Submission/` final packaged Track 1 submission
- `Project_432/` Assignment 2 copy of the BlindDrop project and project-level docs
- `script_*.py` and `update_report_cells.py` helper scripts for report and evidence maintenance

## Recommended Reading Order

1. `Project_Assignments/Assignment1/README.md`

## GitHub Upload

Use the following commands from the repository root:

```powershell
git add .
git commit -m "Update Project_Assignments paths"
git push origin main
```

If your branch is not `main`, replace it with your branch name.
